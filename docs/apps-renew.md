# Apps renewal status

Snapshot of every Aomi app's pipeline state. Updated 2026-05-10.

Columns:
- **OpenAPI gen?** — has the spec been drafted/discovered and a Rust client generated by `aomi-build gen-client`? `ext` = lives under `ext/src/<app>/` (shared library); `app-local` = lives under `apps/<app>/src/client/`.
- **Tool compose?** — has `apps/<app>/src/tool.rs` been curated for user-centric naming, composition, and a domain-specific PREAMBLE (per the `aomi-app-ux-tool-maker` skill)?
- **Tested?** — runtime e2e coverage in `aomi/crates/runtime/tests/`. Most apps are only verified at compile-time today.

- **1 app with full e2e test coverage**: khalani.
